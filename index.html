Auditable cell-type annotation, in your browser.

Drop a single-cell RNA-seq dataset (.h5ad) and get back per-cell predicted cell types with the marker-gene evidence behind every call. Built for computational biologists who need defensible labels — not just "the model said so" but "these four genes co-fired and that 4-tuple is associated with cell type X."

  • Audit-trail predictions. Each cell carries the marker tuple that drove its label.
  • Five reference atlases bundled. Human blood / lung / multi-tissue, mouse olfactory bulb / multi-region brain.
  • Annotation in seconds. ~10 MB ONNX model runs locally via WebAssembly.
Private by design. Your file is parsed and analysed entirely in this browser tab. Nothing is uploaded to any server.
